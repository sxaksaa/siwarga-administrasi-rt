# Backend Administrasi RT

REST API Laravel untuk autentikasi, penghuni, rumah, riwayat hunian, tagihan, pembayaran, pengeluaran, dan laporan keuangan.

Panduan instalasi lengkap tersedia di [`../dokumentasi/PANDUAN_INSTALASI.md`](../dokumentasi/PANDUAN_INSTALASI.md).

Perintah utama:

```powershell
composer install
php artisan migrate --seed
php artisan serve
php artisan test
```
