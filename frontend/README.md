# Frontend Administrasi RT

Antarmuka React untuk aplikasi Administrasi RT. Seluruh data berasal dari REST API Laravel pada folder `backend`.

Panduan instalasi lengkap tersedia di [`../dokumentasi/PANDUAN_INSTALASI.md`](../dokumentasi/PANDUAN_INSTALASI.md).

Perintah utama:

```powershell
npm install
npm run dev
npm run build
```
